🔧 Quick Setup Guide
1️⃣
Install Ollama
Download from ollama.com and install it on your computer
ollama pull llama3
2️⃣
Start Ollama
Open a terminal window and start the Ollama server
ollama serve
3️⃣
Start Nobichan
In another terminal, run the Flask app
python app.py
